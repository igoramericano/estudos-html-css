Trabalhando com listas

Listas ordenadas

a tag ol se refere a order list, ou lista ordenada, para numerar coisas, passos ou tarefas, etapas etc.

  1. Acordar
  2. Tomar café
  3. Tomar banho
  4. F1
  5. Estudar
  6. Jogar

Listas não ordenadas

a tag ul se refere a unorder list, ou lista não ordenada, para listar coisas, itens, produtos etc.

  • Arroz
  • Feijão
  • Carne
  • Batata
  • Salada
