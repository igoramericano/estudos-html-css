Grid Layout parte 3

Grid-auto-columns e grid-auto-rows são propriedades do CSS Grid Layout que permitem definir o tamanho automático das colunas e linhas quando o conteúdo excede o espaço disponível.

Grid-auto-columns define o tamanho automático das colunas

Grid-auto-rows define o tamanho automático das linhas

Grid-auto-columns e grid-auto-rows são úteis para criar layouts responsivos, onde o tamanho das colunas e linhas se ajusta automaticamente com base no conteúdo.
